# yii2-phpexcel

[PHPExcel](https://github.com/PHPOffice/PHPExcel) response formatter for Yii2.

https://packagist.org/packages/iutbay/yii2-phpexcel

Installation
------------
The preferred way to install this response formatter is through [composer](http://getcomposer.org/download/).

Either run

```
composer require "iutbay/yii2-phpexcel" "*"
```

or add

```json
"iutbay/yii2-phpexcel" : "*"
```

to the require section of your application's `composer.json` file.
